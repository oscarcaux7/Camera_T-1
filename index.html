Camera T-1

The Camera T-1 is a tangible device designed to "break the temporal frame" by fostering emotional engagement with historical environments.

Core Concept: Temporal Exploration

The project aims to bridge individual and collective memory through an interactive, physical experience.

  • In-situ Retrieval: Users can rediscover past photographs of their current location through the camera's screen.
  • Archival Window: These images serve as "windows" into the lives of previous generations, offering a glimpse into how they experienced the world.
  • Database: It leverages an extensive database of over 19,000 photos of Paris, dating from 1850 to 2024.

Technical Architecture

The device is a complete hardware-software system built for research and interactive demos.

  • Hardware Core: Integrated a Raspberry Pi 4 card with a camera module and an RGB touch screen.
  • Power Management: Engineered a robust power system using a TPS563200 to transform 7.4V from an 18650 battery holder into a stable 5V 3A.
  • Embedded Software: Developed a custom UI using Python (Tkinter) to manage real-time location data and Wi-Fi for image retrieval.
  • Physical Prototyping: Features a custom 3D-printed enclosure with tactile peripheral management like rotary encoders.

Academic Context

This project is currently being developed for the CHI 2026 Student Research Competition.

  • Research Goal: Explore how tangible "time telescopes" can prompt empathy for the present environment.
  • SOTA Inspiration: Building upon related work such as Paragraphica (location-based generation) and Chronoscope (non-linear memory exploration).
